Use a simple in-memory Map with TTL (15 minutes) keyed by `${workspaceId}:${reportType}:${optionsHash}`. Do NOT create a new database table -- in-memory cache is sufficient for single-instance Next.js. Export: `getCachedAnalytics<T>(key: string): T | null`, `setCachedAnalytics<T>(key: string, data: T, ttlMs?: number): void`, `invalidateAnalyticsCache(workspaceId: string): void`. Default TTL 15 minutes.

**win-loss.ts**: Upgrade `getWinLossPatterns()` to add signal-events-based patterns while preserving all existing patterns (deal size, velocity, engagement by notes). Add these new pattern analyses:

1. **Stakeholder engagement pattern**: Query `signal_events` for closed deal record_ids, count distinct signal types per deal (emails, meetings, calls). Group into "low engagement" (0-3 signals), "moderate" (4-10), "high" (11+). Compare win rates across buckets. Finding: "Deals with 10+ touchpoints close at X% vs Y% overall".

2. **Meeting cadence pattern**: Count `meeting_ended` signals per closed deal. Compare win rates for deals with 0 meetings vs 1-2 vs 3+.

3. **Response time pattern** (if email signals exist): For deals with `email_received` signals, check time gap between consecutive signals. Deals where average gap < 3 days vs > 7 days -- compare win rates.

For each pattern, only include if the bucket has 3+ deals (existing minimum). Keep the existing 3 patterns (deal size, velocity, notes engagement). Cap total patterns at 7, sorted by the most statistically interesting (largest deviation from overall win rate).

Wrap the entire `getWinLossPatterns` result in the cache: check cache first, return if fresh, compute and cache if stale.

Update the AI narrative prompt to mention the new signal-based patterns alongside existing ones. Keep the existing OpenRouter call pattern (non-streaming, workspace config resolution).

Preserve the existing `WinLossPattern` and `WinLossAnalysis` interfaces -- the frontend components already consume these. No interface changes needed since patterns are just additional entries in the patterns array.
